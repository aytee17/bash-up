# up
Quickly move up directories.

## Usage

![usage](https://i.imgur.com/GgqZXjC.gif)

## Installation
Move the script to a directory in your ```PATH``` 

```sudo mv up.sh /usr/local/bin```

Then add the following line to ```~/.bashrc```

```. /usr/local/bin/up.sh```

Reload the file

```. .bashrc```

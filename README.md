# up
Quickly move up directories.

## Usage
```
usage: up [levels] 
  levels: the number of directories to go up
```
 
<img src="https://i.imgur.com/GgqZXjC.gif" width="600" height="348" />

## Installation

Source the script in ```~/.bashrc``` 

    . /path/to/up.sh

Reload the file

    . .bashrc
